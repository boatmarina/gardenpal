Claude MCP access

Generate a personal token, then add the MCP server to Claude Desktop so Claude can read and update your garden tracker directly from any chat — no copy-pasting needed.

{% if user.api_token %}

Your token

{{ user.api_token }}

Claude Desktop config

Add this to your claude_desktop_config.json (find it via Claude Desktop → Settings → Developer): 

{
  "mcpServers": {
    "gardenpal": {
      "command": "python",
      "args": ["/path/to/gardenpal/mcp/server.py"],
      "env": {
        "GARDENPAL_API_URL": "{{ request.host_url.rstrip('/') }}",
        "GARDENPAL_API_TOKEN": "{{ user.api_token }}"
      }
    }
  }
}

Replace /path/to/gardenpal with where you cloned the repo. Install deps once: pip install httpx "mcp[cli]" then restart Claude Desktop. Once connected, just tell Claude: "Add zucchini to my garden tracker, planted today in raised bed 1."
